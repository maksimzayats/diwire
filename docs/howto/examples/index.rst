Examples (runnable)
===================

All examples below live in the repository's ``examples/`` folder and can be run directly:

.. code-block:: bash

   uv run python examples/ex01_basics/ex01_registration.py

Learning path
-------------

1. Basics (registration, lifetimes, constructor injection)
2. Scopes (SCOPED, generator cleanup, nested scopes)
3. Function injection (Injected marker)
4. Patterns (request handlers, repositories, interfaces)
5. Async
6. FastAPI
7. Errors (what failures look like)
